# E-Commerce Sales Analysis Dashboard using SQL & Excel

## Tools Used
- Oracle SQL
- Microsoft Excel

## Project Description
Developed an E-Commerce Sales Analysis Dashboard using Oracle SQL and Excel to analyze customer orders, sales trends, product performance, and delivery status.

## Features
- Sales performance analysis
- Customer order tracking
- Product category analysis
- Monthly revenue reporting
- Delayed delivery monitoring
- Excel dashboard visualization

## SQL Concepts Used
- JOINS
- GROUP BY
- ORDER BY
- Aggregate Functions
- Subqueries
- CASE Statements

## Dashboard Insights
- Highest selling products
- Top revenue generating categories
- Monthly sales trends
- Delivery performance tracking

## Outcome
Improved business reporting and operational analysis by generating structured insights from sales and order data.
